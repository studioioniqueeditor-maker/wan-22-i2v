# Technology Stack: VividFlow

## Core Backend
*   **Language:** Python 3.8+
*   **Framework:** Flask (Lightweight WSGI web application framework)
*   **API Client:** `requests` (For synchronous communication with RunPod's serverless API)

## Frontend & UI
*   **Templating:** Jinja2 (Integrated with Flask for server-side rendering)
*   **Styling:** HTML5/CSS3 (Planned upgrade to Tailwind CSS and modern UI components)

## Infrastructure & Services
*   **GPU Compute:** RunPod Serverless (Wan 2.1 I2V Model)
*   **Environment Management:** `python-dotenv` (For secure handling of API keys and endpoint IDs)
*   **Storage:** Google Cloud Storage (GCS) for persistent video storage

## Planned Additions
*   **Prompt Enhancement:** Groq API (Integration for fast LLM-based prompt expansion)
